Web server : A computer program that accepts HTTP requests from clients, such as web browsers , and returns Web pages, such as HTML documents

A web server refers to a service program that transmits HTML documents or objects (image files, etc.) requested by a web browser through HTTP or HTTPS. Sometimes the hardware that runs web server software is also confused with a web server.
